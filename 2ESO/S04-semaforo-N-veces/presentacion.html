Semáforo de N ciclos

Bucle FOR: el programa hace algo un número exacto de veces y luego para

⚡ Tinkercad 🔁 FOR 🔴 LED pin 11 🟡 LED pin 12 🟢 LED pin 13 ⏱ 55 min

Concepto: FOR vs loop() infinito

♾️ loop() — infinito

El código dentro de void loop() se ejecuta una y otra vez sin parar hasta que se desconecta la placa.

Útil cuando quieres una respuesta continua (semáforo eterno).

🔁 for — con final

for (int i = 0; i < N; i++) { ... }

Ejecuta el bloque N veces y después continúa con el siguiente código. Tiene un final programado.

💡 Hoy queremos un semáforo que se apague solo: hace 5 ciclos completos y se detiene. Ponemos el for dentro de setup() (se ejecuta una sola vez al arrancar) y dejamos loop() vacío.

El reto de hoy 🎯 Semáforo de N ciclos

📋 ¿Qué tienes que hacer?

Monta un semáforo con 3 LEDs (rojo, ámbar, verde) y haz que recorra su ciclo completo 5 veces. Cuando termine, los tres LEDs deben quedar apagados.
👉 Rojo 1 s · Ámbar 0,5 s · Verde 1 s · Ámbar 0,5 s. Ese ciclo se repite 5 veces.
Usa un for (int i = 0; i < 5; i++) dentro de setup().

El código con FOR 💻

const int rojo = 11;
const int ambar = 12;
const int verde = 13;
 
void setup() {
pinMode(rojo, OUTPUT);
pinMode(ambar, OUTPUT);
pinMode(verde, OUTPUT);
 
for (int i = 0; i < 5; i++) {
digitalWrite(rojo, HIGH); delay(1000); digitalWrite(rojo, LOW);
digitalWrite(ambar,HIGH); delay(500); digitalWrite(ambar,LOW);
digitalWrite(verde,HIGH); delay(1000); digitalWrite(verde,LOW);
digitalWrite(ambar,HIGH); delay(500); digitalWrite(ambar,LOW);
}
}
 
void loop() { // vacío }
