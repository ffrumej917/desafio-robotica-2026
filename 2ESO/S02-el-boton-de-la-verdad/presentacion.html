El Botón de la Verdad

IF con pulsador real: el LED solo enciende si se cumple la condición

⚡ Tinkercad 🔀 IF (condicional) 🔘 Pulsador pin 7 💡 LED pin 13 ⏱ 55 min

Concepto: IF + pulsador

🔘 Pulsador (entrada digital)

Un pulsador conectado al pin 7 con resistencia pull-down de 10 kΩ. Lee HIGH (1) cuando está pulsado, LOW (0) cuando no.

💡 LED (salida digital)

Un LED en el pin 13 con resistencia de 220 Ω. Recibe HIGH para encender y LOW para apagar.

⚠️ Resistencia pull-down (10 kΩ): sin ella, el pin 7 flota entre 0 y 1 de forma aleatoria cuando el botón no está pulsado. La resistencia pull-down fuerza la lectura a LOW (0) cuando el botón está abierto, garantizando lecturas fiables.

El reto de hoy 🎯 El Botón de la Verdad

📋 ¿Qué tienes que hacer?

Monta el circuito en Tinkercad con un pulsador y un LED. Programa el Arduino para que:
👉 SI el pulsador está pulsado (pin 7 = HIGH) → enciende el LED (pin 13 = HIGH).
👉 Si no está pulsado → el LED permanece apagado.
Usa la estructura E/D/P: Entender, Diseñar, Programar.

El código que necesitas 💻

// Definir pines
const int pinBoton = 7;
const int pinLed = 13;
 
void setup() {
pinMode(pinBoton, INPUT);
pinMode(pinLed, OUTPUT);
}
 
void loop() {
if (digitalRead(pinBoton) == HIGH) {
digitalWrite(pinLed, HIGH);
}
}
