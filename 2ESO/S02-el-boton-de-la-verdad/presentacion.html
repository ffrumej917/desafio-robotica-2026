🔘 El botón de la verdad

Aprendemos a leer una entrada digital con digitalRead() y a tomar decisiones con IF.

DESAFÍO ROBÓTICA 2026 · IES Punta del Verde

⚡ El circuito

Componentes y conexiones

Pin 13 → LED

Resistencia 220 Ω en serie. Salida digital (OUTPUT).

Pin 7 → Pulsador

Entrada digital (INPUT). Pull-down 10 kΩ a GND.

Pull-down 10 kΩ

Garantiza que pin 7 = LOW cuando el botón está suelto. Sin ella: estado flotante.

💻 El código

Estructura IF en Arduino (C++)

Dos IF independientes — sin ELSE todavía (eso es S03).

const int PIN_LED = 13; const int PIN_BOTON = 7; void setup() { pinMode(PIN_LED, OUTPUT); pinMode(PIN_BOTON, INPUT); } void loop() { if (digitalRead(PIN_BOTON) == HIGH) { digitalWrite(PIN_LED, HIGH); // pulso → LED ON } if (digitalRead(PIN_BOTON) == LOW) { digitalWrite(PIN_LED, LOW); // suelto → LED OFF } }
