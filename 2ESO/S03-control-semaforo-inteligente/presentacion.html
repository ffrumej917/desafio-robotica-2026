Control de Semáforo Inteligente

IF-ELSE anidado con interruptor, pulsador y variable bool

⚡ Tinkercad 🔀 IF-ELSE anidado 🚦 3 LEDs 🎚 Interruptor + pulsador 📌 Variable bool ⏱ 55 min

Dos componentes y una variable nueva

🎚 Interruptor (pin 6)

Encendido/apagado del semáforo. Mantiene su estado físico (HIGH o LOW estable). No hay que mantenerlo pulsado: lo dejas en una posición y se queda ahí.

🔘 Pulsador (pin 7)

Activa la secuencia de emergencia. Solo está en HIGH mientras lo aprietas. Al soltar vuelve a LOW.

📌 Variable bool (booleana): guarda solo dos valores posibles, true o false. La usaremos como un post-it: "¿se ha pulsado el botón en algún momento del ciclo?". Mientras dura el delay, vamos comprobando el pulsador y, si se pulsa, escribimos true en la variable. Al final del ciclo miramos el post-it para decidir si activamos la emergencia.

El reto 🎯 Control de Semáforo Inteligente

📋 Comportamiento esperado

Interruptor (pin 6) en OFF: los 3 LEDs apagados.
Interruptor en ON: ciclo verde (1 s) → amarillo (1 s) → rojo (1 s) → repite.
Pulsador (pin 7): si lo pulsas en cualquier momento del ciclo, al terminar el rojo se activa la secuencia de emergencia: amarillo durante 2 s. Después vuelve al ciclo normal.

3 LEDs (verde·pin 13 / amarillo·pin 12 / rojo·pin 11) · interruptor·pin 6 · pulsador·pin 7 · pull-downs 10 kΩ · resistencias 220 Ω

El código 💻

void setup() {
pinMode(13, OUTPUT); // LED verde
pinMode(12, OUTPUT); // LED amarillo
pinMode(11, OUTPUT); // LED rojo
pinMode(6, INPUT); // interruptor
pinMode(7, INPUT); // pulsador
}
 
void loop() {
if (digitalRead(6) == HIGH) {
bool emergencia = false;
 
// Verde 1 s
digitalWrite(13, HIGH);
if (digitalRead(7) == HIGH) emergencia = true;
delay(1000);
digitalWrite(13, LOW);
 
// Amarillo 1 s
digitalWrite(12, HIGH);
if (digitalRead(7) == HIGH) emergencia = true;
delay(1000);
digitalWrite(12, LOW);
 
// Rojo 1 s
digitalWrite(11, HIGH);
if (digitalRead(7) == HIGH) emergencia = true;
delay(1000);
digitalWrite(11, LOW);
 
// IF anidado
if (emergencia == true) {
digitalWrite(12, HIGH);
delay(2000);
digitalWrite(12, LOW);
}
} else {
digitalWrite(13, LOW);
digitalWrite(12, LOW);
digitalWrite(11, LOW);
}
}
