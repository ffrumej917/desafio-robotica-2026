Aprende IF-ELSE: cuando hay DOS caminos posibles

🐱 Scratch 🔀 IF-ELSE ⌨️ Bloque preguntar ⏱ 55 min

¿Qué es el IF-ELSE?

SI (IF) ✅

Si la condición es verdadera, el programa ejecuta este bloque de instrucciones.

SI NO (ELSE) ❌

Si la condición es falsa, el programa ejecuta este otro bloque en su lugar.

🎯 Analogía real: Tu teléfono te pide el PIN. Si es correcto → desbloquea. Si no → te lo deniega. ¡Eso mismo vas a programar tú hoy!

El reto de hoy 🎯 El Robot Portero

📋 ¿Qué tienes que hacer?

Crea un sprite en Scratch que actúe como portero. Al hacer clic, el robot pregunta una contraseña.
👉 Si el usuario escribe "abre" → el sprite dice "¡Puedes pasar! ✅" y cambia de disfraz o fondo.
👉 Si escribe cualquier otra cosa → dice "¡Acceso denegado! 🚫" y reacciona (giro, cambio de color…)
