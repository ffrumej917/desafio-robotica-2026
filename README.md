# Desafío Robótica 2026

**Curso práctico de Computación y Robótica para 1º y 2º ESO** · IES Punta del Verde · Tercer trimestre 2025-26.

Entrenamiento de programación y pensamiento computacional como preparación para el concurso **FIRST LEGO League** del curso siguiente. Dos cursos en paralelo con el mismo esqueleto conceptual pero distinta plataforma:

- **1º ESO** trabaja con **Scratch** (programación visual pura, sprite como "robot").
- **2º ESO** trabaja con **Tinkercad** (componentes electrónicos simulados, introducción a Arduino).

## 🎯 Objetivos del bloque

- Consolidar estructuras de programación (secuencia, condicional, bucle).
- Introducir el ciclo sensor–controlador–actuador.
- Promover trabajo en equipo y resolución autónoma.
- Preparar habilidades transferibles para competiciones de robótica (FLL, SPIKE Prime).
